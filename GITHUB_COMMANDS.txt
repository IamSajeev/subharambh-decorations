## üîß COMMANDS TO RUN IN POWERSHELL

### Step 1: Copy the command below and paste into PowerShell

```powershell
cd c:\Users\sajee\subharambh-decorations
```

Press Enter.

---

### Step 2: Run this (to set main branch)

```powershell
git branch -M main
```

Press Enter.

---

### Step 3: Run this (IMPORTANT: Replace YOUR_USERNAME!)

**Before you run, replace `YOUR_USERNAME` with your GitHub username!**

Example: If your username is "john-smith", use: `john-smith`

```powershell
git remote add origin https://github.com/YOUR_USERNAME/subharambh-decorations.git
```

Press Enter.

---

### Step 4: Verify the remote was added

```powershell
git remote -v
```

Press Enter.

Should show:
```
origin  https://github.com/YOUR_USERNAME/subharambh-decorations.git (fetch)
origin  https://github.com/YOUR_USERNAME/subharambh-decorations.git (push)
```

---

### Step 5: Add all files

```powershell
git add .
```

Press Enter.

---

### Step 6: Commit with message

```powershell
git commit -m "Initial commit - Subharambh Decorations professional website"
```

Press Enter.

---

### Step 7: Push to GitHub (FIRST TIME ONLY)

```powershell
git push -u origin main
```

Press Enter.

**May ask for GitHub username & password** (or Personal Access Token).

---

### Step 8: Verify success

```powershell
git log --oneline -n 5
```

Press Enter.

Should show your commit.

---

## ‚úÖ AFTER PUSH IS COMPLETE

### Check on GitHub:
1. Go to `github.com/YOUR_USERNAME/subharambh-decorations`
2. You should see all your files
3. Refresh if needed

### Enable GitHub Pages:
1. Click **Settings**
2. Click **Pages** (left sidebar)
3. Select Branch: **main**
4. Select Folder: **/ (root)**
5. Click **Save**
6. Wait 1-2 minutes
7. Your website will be live! üéâ
